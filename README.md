# Automatic Video Captioning Using Audio-Visual Clues

## Overview

This project generates automatic captions for videos by combining **audio cues** and **visual clues** using machine learning and video processing techniques.

## Features

* Extracts information from video frames
* Uses audio signals for contextual understanding
* Generates automatic captions
* Multimodal approach using audio + visual inputs

## Tech Stack

* Python
* OpenCV
* MoviePy
* Machine Learning / Deep Learning

## Files

* `main.py` – Main execution file
* `main2.py` – Supporting model logic
* `main3.py` – Additional processing
* `test_moviepy.py` – Testing video processing
* `input.mp4` – Sample input video

## Applications

* Video accessibility
* Content summarization
* Smart subtitles
* Assistive technologies

## Future Improvements

* Real-time caption generation
* Transformer-based captioning models
* Higher contextual accuracy
